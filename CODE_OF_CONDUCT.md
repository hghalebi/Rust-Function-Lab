# Contributor Covenant Code of Conduct

## Our Pledge

We are committed to a welcoming and respectful environment for everyone.

## Standards

Examples of behavior that contributes to a positive environment include:
- Being respectful of differing viewpoints
- Using inclusive language
- Accepting constructive feedback

Examples of unacceptable behavior include:
- Harassment, intimidation, or discrimination
- Trolling, insults, or personal attacks
- Publishing private information without permission

## Scope

This Code of Conduct applies to project spaces: repository issues, pull requests,
and discussion channels connected to this repository.

## Enforcement

If you experience or witness bad behavior, open an issue or email the maintainer.
If a report is confirmed, maintainers may:
- issue a warning
- remove offending content
- ban contributions from repeat offenders

Thanks for helping us keep this space professional and safe.
